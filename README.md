# Niche Navigator

A personalized discovery tool for people new to tech and web3 to find their ideal career niche.

## Framework

Vanilla JavaScript with Tailwind CSS (CDN)

## Preview

Configured to run on `0.0.0.0:8080` for k8s HTTPRoute access.

## Theme

Clean Light theme with white backgrounds, soft gray accents, and blue primary color. Modern, professional design with subtle shadows and rounded corners.

## Features

- Interactive 8-question quiz
- 20 tech and web3 niche recommendations
- Personalized matching algorithm
- Detailed niche information including salary ranges, difficulty levels, and required skills
- Progress tracking
- Results sharing functionality
- Next steps and learning resources
